Welcome!

epydial is a little application in order to test all the brilliant pyneo daemons. the purpose of this development is to integrate all functions of pyneo in one application all together. both gta01 (neo1973) and gta02 (freerunner) are used as test devices. But the main focus is on the functionality of the telephon (dialer, sms, etc.)

I want it now @ gitorious!

About

epydial is a graphical user interface for the Openmoko Neo GTA01 and Freerunner (GTA02). As background it uses the brilliant pyneo demons. It was planed as a test application for the pyneo mobile softwarestack - however it grows and grows :)

Navigation

The navigation is a bit based on that of a normal feature phone. You can navigate through the applications with shortcuts - for example you enter 1 - touch the statusbar and you will start the gsm application with a few configuration options. With a back button you will get to the main menu. Many applications are hidden behind the numbers - test it :)

Application

So far we have many applications like a dialer, sms, contacts, mapviewer, hot or not, picture viewer ... and many more - check out the screenshot section with all applications :)

Base

epydial is based on python (Since 2012 we also use C) and uses the enlightenment graphic library - to cut a long story shot - evas and edje. We think that edje is a bit heavy for a mobile device so we will port all applications that uses edje to evas.
